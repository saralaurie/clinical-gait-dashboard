# Clinical Gait Triage Dashboard (FHIR-Enabled)

An interoperable dashboard designed for interdisciplinary geriatric rehab teams to track **Gait Speed as the 6th Vital Sign.**

### Technical Overview
* **Standard:** HL7 FHIR (R4)
* **Data Source:** HAPI FHIR Public Sandbox
* **Logic:** Automated triage alerts for patients below the **0.6 m/s** high-risk threshold.

### Disclaimer
This project uses a public sandbox environment. Because the data is live and open-access, patient records may be modified by other users. Please refer to the [Loom Video](https://www.loom.com/share/194b495991de416b82b6e98f17c0c814) for the full clinical demonstration.
